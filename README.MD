## Описание проекта

## Функциональные возможности

### Для пользователей:
- Просмотр каталога продуктов
- Поиск и фильтрация продуктов по категориям
- Добавление продуктов в корзину
- Оформление заказов
- Оставление отзывов
- Управление профилем

### Для администраторов:
- Управление пользователями
- Управление продуктами
- Модерация отзывов
- Управление категориями и единицами измерения

## Технологии

- Python 3.11+
- FastAPI
- SQLAlchemy
- Alembic
- SQLite
- Jinja2
- HTML/CSS/JavaScript
- Font Awesome

## Установка

1. Клонируйте репозиторий:
```bash
git clone <url_репозитория>
cd <имя_папки_проекта>
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # На Linux/Mac
# или
venv\Scripts\activate  # На Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```


4. Запустите приложение:
```bash
python main.py
```

## Настройка

### Конфигурация базы данных

В файле `.env` укажите следующие параметры:

- `SECRET_KEY` - секретный ключ для JWT токенов
- `ALGORITHM` - алгоритм шифрования для JWT токенов
- `ACCESS_TOKEN_EXPIRE_MINUTES` - время жизни токена в минутах
- `DB_NAME` - имя файла базы данных (для SQLite) или название базы данных

## API документация

После запуска приложения документация к API доступна по адресу:
- `/docs` - интерактивная документация Swagger
- `/redoc` - документация ReDoc

## Структура проекта

```
app/
├── api/          # Роутеры FastAPI
├── models/       # Модели SQLAlchemy
├── repositories/ # Репозитории для работы с базой данных
├── services/     # Бизнес-логика
├── schemes/      # Pydantic схемы
├── exceptions/   # Исключения
├── database/     # Конфигурация базы данных
├── static/       # Статические файлы
├── templates/    # Шаблоны Jinja2
└── config.py     # Конфигурация приложения
```

## Миграции

Для создания новой миграции:
```bash
alembic revision --autogenerate -m "Описание миграции"
```

Для применения миграций:
```bash
alembic upgrade head
```
